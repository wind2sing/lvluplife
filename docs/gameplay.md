# 玩法与机制

## 核心循环

1. 从任务公会、每日冒险板或自己的计划中选择一项现实任务。
2. 在现实中真正完成它。
3. 写下过程，也可以上传图片或附件。
4. 获得经验和六项属性成长。
5. 提升等级，逐步发现迷雾后的新成就。

应用不会替你判断一件事是否真正发生。荣誉规则只有一条：完成现实行动后再领取奖励。

## 等级与迷雾

- 等级 1 升到等级 2 需要 500 经验。
- 此后每一级所需经验增加 180。
- 等级不足的任务只显示迷雾，不泄露名称、描述和奖励，也不能点击进入。
- 撤销完成记录会扣回对应经验，等级可能下降。

## 六项属性

| 代码 | 属性 | 代表内容 |
| --- | --- | --- |
| STR | 力量 | 健康、体能、运动与身体行动 |
| CUL | 文化 | 艺术、历史、传统与开阔眼界 |
| ENV | 环境 | 家庭、户外、城市与周围世界 |
| CHA | 魅力 | 社交、表达、沟通与人际关系 |
| TAL | 才能 | 技能、创造力、专业能力与手作 |
| INT | 智慧 | 学习、思考、研究与解决问题 |

属性用于描述成长方向，不评价人的价值。专精只影响每日推荐倾向，不增加经验或属性优势。

## 奖励规则

原版挑战、自动模式个人任务和计划步骤使用统一预算：

- 所需精力影响经验和属性预算。
- 高频循环任务奖励低于终身一次成就，避免刷分。
- 任务等级提供小幅修正。
- 精力与重复周期用于推导任务稀有度。
- 属性奖励根据分类对应的主、副属性分配。
- 系统不猜测任务需要多少分钟。

完成记录会保存当次奖励快照，之后调整公式不会改变已经获得的历史经验和属性。

## 重复、撤销与封印

| 周期 | 再次获得奖励的等待时间 |
| --- | --- |
| 每日 | 1 天 |
| 每周 | 7 天 |
| 每月 | 30 天 |
| 每年 | 365 天 |
| 终身一次 | 只能获得一次奖励 |

- 已接取任务可以退回。
- 已完成任务可以撤销，经验、属性、冷却和附件会同步回退。
- 暂时不想看见的任务可以封印，并随时从封印库恢复。

## 行动力与每日推荐

行动力不是生命值，而是限制短时间集中刷取奖励的节奏机制。它不会造成失败或惩罚。

每日冒险板提供三个候选：轻松一胜、成长任务和自由选择。推荐会考虑当前精力、等级、已接任务、收藏、属性短板、最近完成分类和冷却状态，并显示推荐理由。

## 完成记录与附件

- 每次完成可记录最多 280 字。
- 最多上传 3 个附件，单个不超过 10 MB。
- 支持常见图片、PDF、Office、文本、Markdown 和 ZIP。
- 图片直接渲染，其他附件可下载。
- 撤销完成时会删除对应的私密 Blob 文件。

## 私人收藏馆

收藏馆包含称号、徽章、头像框、营地主题和纪念物。解锁进度来自真实完成次数、日期、属性、附件、任务链和特殊条件。

收藏馆可以在设置中整体关闭；关闭只隐藏入口并暂停装扮效果，不会删除进度或装备选择。
