# 玩法与机制

## 核心循环

1. 从任务公会、每日冒险板或自己的计划中选择一项现实任务。
2. 在现实中真正完成它。
3. 写下过程，也可以上传图片或附件。
4. 获得经验和六项属性成长。
5. 提升等级与分类探索阶段，逐步发现迷雾后的新成就。

应用不会替你判断一件事是否真正发生。荣誉规则只有一条：完成现实行动后再领取奖励。

## 任务描述

538 项原版挑战都包含独立的中英文任务描述。描述会结合任务标题、所属分类和探索阶段，用简短的 RPG 委托语气补充行动氛围，例如把家务称为“营地维护”、把学习称为“求知支线”、把旅行称为“远征”。

- 描述只负责增加趣味和提供观察角度，不会偷偷增加完成条件。
- 切换界面语言时，任务标题和描述会一起切换。
- 已发现任务的描述可以参与任务公会搜索；迷雾任务仍不会通过搜索泄露内容。
- 描述由挑战数据生成脚本稳定生成，相同任务每次重新导入都会得到相同文案。
- 自定义任务仍可由用户填写自己的描述；留空时使用对应重复周期的通用说明。

## 等级、分类探索与迷雾

- 等级 1 升到等级 2 需要 500 经验。
- 此后每一级所需经验增加 180。
- 总等级决定任务是否具备领取资格，分类探索决定具体发现哪些任务。
- 探索型分类只会先出现少量入门任务，高等级角色新进入一个分类时也不会一次解锁全部内容。
- 完成不同任务会提高分类阶段并发现新内容；循环完成同一个任务不会重复推进。
- 封印不适合的任务会补充新的发现，避免被当前候选卡住。
- 已发现任务永久保留；撤销完成可能降低总等级和领取资格，但不会让任务重新变成迷雾。
- 尚未发现的任务不泄露名称、描述和奖励，也不能点击进入。
- 撤销完成记录会扣回对应经验，等级可能下降。

### 分类探索的精确规则

每个分类的发现目标数按以下公式计算：

```text
发现目标数 = 4 + 不同任务完成数 × 2 + 当前分类封印数
```

发现目标数只是候选上限，实际发现仍需要同时满足任务的分类 Tier 与总等级，并且不会超过分类内的任务总数。名额不足时，系统按照原挑战列表顺序发现，保证原版内容的渐进关系稳定。

| 条件 | 结果 |
| --- | --- |
| 第一次进入分类 | 最多发现 4 个入门任务 |
| 完成 1 个不同任务 | 开放 Tier 2，每项不同完成增加 2 个发现名额 |
| 完成 3 个不同任务 | 开放 Tier 3 |
| 完成 10 个不同任务 | 开放 Tier 4 |
| 封印当前分类任务 | 每封印 1 项，补充 1 个发现名额 |
| 循环完成同一任务 | 可以再次获得周期奖励，但不重复推进分类探索 |
| 任务已经发现 | 写入 `discoveredIds`，之后永久保留 |

例如：

- 等级 32 的角色第一次进入“运动”，仍处于“初识”，只发现 4 / 16 项，而不是因为总等级高就显示全部任务。
- 在“摄影”完成 3 个不同任务后进入“熟练”，发现目标数为 `4 + 3 × 2 = 10`，因此可发现 10 / 17 项；更深任务还要满足自身总等级要求。

不同可见状态的行为如下：

- **已发现且等级足够：** 可以查看详情、接取和完成。
- **已发现但等级不足：** 显示名称、描述和奖励，可以查看详情，但不能接取。
- **尚未发现：** 只显示不可点击的迷雾，不可搜索、不可通过地址直达，也不会进入每日推荐。

自定义任务和任务链 / 项目步骤会自动发现。旧存档升级时，已接取、收藏、封印或完成过的任务会自动写入已发现列表，避免迁移后重新变成迷雾。撤销完成可能降低分类完成数、总经验和领取资格，但已发现内容不会回退。

### 图鉴型目录：经典书单

经典书单是分类探索的明确例外。它不是按难度排列的任务线，而是一份可以自由选择的完整书目：

- 150 本全部为等级 1，并从一开始完整显示。
- 每本书都是终身一次成就，统一使用支线任务奖励档。
- 页面显示已读进度，并支持使用书名、作者或描述搜索。
- 书单任务不会自动占据每日推荐；收藏或接取后才可能被推荐。
- 完成书单不会推进其他分类的探索阶段。

## 六项属性

| 代码 | 属性 | 代表内容 |
| --- | --- | --- |
| STR | 力量 | 健康、体能、运动与身体行动 |
| CUL | 文化 | 艺术、历史、传统与开阔眼界 |
| ENV | 环境 | 家庭、户外、城市与周围世界 |
| CHA | 魅力 | 社交、表达、沟通与人际关系 |
| TAL | 才能 | 技能、创造力、专业能力与手作 |
| INT | 智慧 | 学习、思考、研究与解决问题 |

属性用于描述成长方向，不评价人的价值。专精只影响每日推荐倾向，不增加经验或属性优势。

## 奖励规则

原版挑战、自动模式个人任务和计划步骤使用统一预算：

- 所需精力影响经验和属性预算。
- 高频循环任务奖励低于终身一次成就，避免刷分。
- 任务等级提供小幅修正。
- 精力与重复周期用于推导任务稀有度。
- 属性奖励根据分类对应的主、副属性分配。
- 系统不猜测任务需要多少分钟。

完成记录会保存当次奖励快照，之后调整公式不会改变已经获得的历史经验和属性。

## 重复、撤销与封印

| 周期 | 再次获得奖励的刷新时间 |
| --- | --- |
| 每日 | 次日 00:00 |
| 每周 | 下周一 00:00 |
| 每月 | 下个月 1 日 00:00 |
| 每年 | 下一年 1 月 1 日 00:00 |
| 终身一次 | 只能获得一次奖励 |

冷却按使用者设备的本地时区计算。例如当天 20:00 完成每日任务，次日 00:00 即可再次完成，不需要等待完整 24 小时。

- 已接取任务可以退回。
- 可循环任务完成后仍保留在“当前任务”；冷却期间可以查看或主动退回，刷新后直接恢复完成入口。
- 终身一次任务完成后会自动离开“当前任务”。
- 已完成任务可以撤销，经验、属性、冷却和附件会同步回退。
- 暂时不想看见的任务可以封印，并随时从封印库恢复。

## 行动力与每日推荐

行动力不是生命值，而是限制短时间集中刷取奖励的节奏机制。它不会造成失败或惩罚。

每日冒险板提供三个候选：轻松一胜、成长任务和自由选择。推荐会考虑当前精力、等级、已接任务、收藏、属性短板、最近完成分类和冷却状态，并显示推荐理由。

## 完成记录与附件

- 每次完成可记录最多 280 字。
- 最多上传 3 个附件，单个不超过 10 MB。
- 支持常见图片、PDF、Office、文本、Markdown 和 ZIP。
- 图片直接渲染，其他附件可下载。
- 撤销完成时会删除对应的私密 Blob 文件。

## 私人收藏馆

收藏馆包含称号、徽章、头像框、营地主题和纪念物。解锁进度来自真实完成次数、日期、属性、附件、任务链和特殊条件。

收藏馆可以在设置中整体关闭；关闭只隐藏入口并暂停装扮效果，不会删除进度或装备选择。
