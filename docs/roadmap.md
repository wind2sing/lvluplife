# 路线图

路线图优先保证单人长期使用的数据可靠性，再扩展玩法。原则是奖励真实行动、允许休息与回头，并避免无意义的货币和数值膨胀。

## P0：可靠性

- [ ] 显示保存中、已同步、失败重试和最后同步时间。
- [ ] 为经验、冷却、行动力、撤销、封印和路由增加自动化测试。
- [ ] 增加完整存档导出、导入和灾难恢复。
- [ ] 为存档结构增加版本号和自动迁移。
- [ ] 增加错误边界和附件上传失败恢复。
- [ ] 选择正式开源许可证。

## P1：玩法增强

### 已完成

- [x] 自定义任务的创建、编辑、复制、封印和奖励计算。
- [x] 每日冒险板、精力状态、换一批和推荐理由。
- [x] 任务链顺序解锁、项目并行步骤和完成总结。
- [x] 七日营火复盘与历史全文搜索。
- [x] 属性阶段、专精、雷达图和私人收藏馆。
- [x] 7 天、30 天、一年统计和月度冒险封面。
- [x] 全局个人创作与收藏馆开关。

### 下一步

- [ ] 编辑完成记录的文字、日期和附件，无需先撤销。
- [ ] 增加“稍后再做”和暂时搁置，到期后重新出现。
- [ ] 为推荐加入“不适合我”“现在没条件”等反馈。
- [ ] 支持自定义分类、图标、颜色和任务模板包。
- [ ] 从 JSON 导入、导出个人任务包。
- [ ] 每周生成跨 2–5 步的主题委托。
- [ ] 区分活跃日与休息日，并支持每周目标频率。
- [ ] 增加六项属性的历史变化时间轴。
- [ ] 增加循环任务坚持率、任务链完成率和封印原因统计。
- [ ] 为终身成就和单条完成记录生成包含日期、文字、照片的纪念卡。

## P2：使用体验

- [x] PWA、应用快捷入口和离线应用外壳。
- [ ] 离线只读存档。
- [ ] 图片灯箱、上传前压缩和更好的附件管理。
- [ ] 任务列表排序：推荐、等级、奖励、周期和最近接触。
- [ ] 键盘快捷键、焦点管理和屏幕阅读器优化。
- [ ] 更多可切换的 RPG 主题。
- [ ] 自动更新桌面端和移动端验收截图。

## 暂不优先

- 好友、社区动态和公开排行榜。
- 强制打卡提醒或惩罚机制。
- 多租户账号系统。

这些功能会显著扩大隐私、审核和运维范围；在单人体验与数据可靠性稳定前不作为近期目标。
